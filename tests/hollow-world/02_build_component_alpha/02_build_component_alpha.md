# Build Component Alpha

Build the first synthetic component of the hollow world project. This task is assigned to a builder agent (hwb1).

## Steps

1. Receive assignment from PM
2. Create the Alpha component artifact: a marker file `alpha.built` in this folder
3. The marker file should contain: agent name, timestamp, and "Component Alpha built successfully"
4. Mark this task complete
5. Report completion to PM

## Acceptance Criteria

- `alpha.built` file exists in `02_build_component_alpha/`
- File contains valid agent name and timestamp
- Task status is `completed`
- PM has been notified
